Setting Up XAMPP

Today we'll be discussing about setting up the Xampp in our personal computers. We'll go through whole process and get Xampp installed in our personal computer.


Introduction to Xampp

XAMPP is a free and open-source cross-platform web server solution stack package developed by Apache Friends, consisting mainly of the Apache HTTP Server,

MariaDB (a fork of MySQL), and interpreters for scripts written in the PHP and Perl programming languages. XAMPP is a useful tool for developers to create and

test web applications locally on their personal computers before deploying them to a live server.





** Setting up Xampp **

The following steps should be followed to successfully setup the Xampp in the PC :



1. Download the Xampp:

i. Visit the official Apache Friends website: Apache Friends.

ii. Choose the version of XAMPP that matches your operating system (Windows, macOS, or Linux).

iii. Click the download link to get the installer.


2. Install the Xampp:

i. Locate the downloaded installer file and double-click it to start the installation process.

ii. We may need to grant permission to run the installer if prompted by your operating system.


3. Configure the Xampp:

i. Start the Control Panel: The XAMPP Control Panel provides an interface to manage the components of XAMPP.

ii. Start Services: In the Control Panel, you will see a list of services: Apache, MySQL, FileZilla, Mercury, and Tomcat.

iii. Click the "Start" button next to "Apache" to start the web server.

iv. Click the "Start" button next to "MySQL" to start the database server.

v. Ensure both services are running (the background of the service name will turn green).


4. Using the Xampp:

i. Place your web application files in the htdocs directory inside the XAMPP installation folder (e.g., C:\xampp\htdocs on Windows).

ii. Access your projects by typing http://localhost/your_project_folder in your web browser.
